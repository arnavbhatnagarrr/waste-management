## 📌 About the Project

Pavitra Foundation is working on solving the growing e-waste crisis through **grassroots logistics software** that enables:

- 📍 Device tracking across lifecycle
- 🔁 Circular economy workflows
- 🤝 Connection between collectors and recyclers
- 📊 Data-driven impact reporting

This website showcases the problem, solution, and mission of the foundation.
